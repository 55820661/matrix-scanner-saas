## Sprint C1.5 - Remote Bootstrap Runtime Completion

Sprint المنفذ:
`Sprint C1.5 - Remote Bootstrap Runtime Completion`

الهدف:
استكمال Remote Bootstrap بحيث يثبت Runtime/Agent الحقيقي القادر على registration وheartbeat وpolling وAgentJob execution وjob result submission، بدل bootstrap demo bundle الذي كان يعمل registration + heartbeat فقط.

ما تم إنجازه:
- تحديث `build_runtime_archive()` ليضم ملفات `scanner_runtime` الحالية داخل tarball بدل bundle مصغر.
- تحديث `agent_service.py` المولد داخل bootstrap archive ليستخدم runtime الحالي من `scanner_runtime.prototype`.
- إضافة loop يقوم بـ:
  - registration عند عدم وجود `agent_token`.
  - حفظ `agent_token` ومسح `registration_token`.
  - heartbeat.
  - polling.
  - تنفيذ job عبر allowlist الحالي.
  - submit result.
- إضافة `runtime_mode = polling_agent` إلى config المرفوع أثناء bootstrap.
- الإبقاء على install path `/opt/matrix_scanner`.
- الإبقاء على systemd service `matrix-scanner-agent.service`.
- الإبقاء على Remote Bootstrap كمسار Matrix Admin-only.
